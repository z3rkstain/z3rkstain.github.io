Why Web Development?

  Over the summer I was in a web development camp at Holberton School in San Francisco. I really enjoyed developing my own website and I wanted to expand my skills by redoing some of the Programming
1-2 and APCS assignments using HTML/CSS/Javascript

What's the Difference?

  Even though Java and Javascript share some simmilarities, web development in general is very different from locally used languages. Using multiple languages and libraries to make a website function is fundamentally different than making compiled programs.

What did I Change?

  A lot of Dave's classworks are very console based or rely on a calling class. For these assignments I decided to make them more visual while retaining functionality. Obviously assignments that relied on calling classes had to be replaced with input boxes.
